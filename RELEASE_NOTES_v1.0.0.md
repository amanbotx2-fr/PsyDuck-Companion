# Ducky v1.0.0

## Highlights

- AI desktop companion with OpenAI, Gemini, Grok, Ollama, and OpenAI-compatible providers
- Smart one-time and recurring reminders
- Daily planner, sticky message, and Pomodoro focus timer
- Searchable AI Model Explorer with favorites and recent models
- Native tray integration, Preferences, eye tracking, and cross-platform desktop packaging
- Automatic update foundation ready for future release publishing

## Supported platforms

- Windows (Setup and MSI)
- macOS (DMG and ZIP)
- Linux (AppImage and DEB)

## Installation

Download the artifact for your platform. On macOS, open the DMG and drag Ducky to Applications. On Windows, run Setup or the MSI installer. On Linux, launch the AppImage or install the DEB package.

## Requirements

Cloud providers require their own API credentials. Ollama works locally with an installed Ollama daemon and does not require a cloud key.
